# Alfred Workflows

Custom Alfred workflows for macOS productivity automation.

## Contents

- **Edge Workspace Switcher** (`ew`) - Quickly switch between Microsoft Edge workspaces
- **Split Display** - Window management for split screen setups

## Installation

Double-click any `.alfredworkflow` file to install. Source files (`.plist`, icons) are included for the Edge Workspace Switcher workflow.

## Usage

Each workflow has its own keyword trigger. Open Alfred and use the workflow's help or explore the workflow settings in Alfred Preferences.
